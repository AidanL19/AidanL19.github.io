Welcome to my final project, Mastermind. Try to guess the random code of colors. In order to guess, you must select a color from the bottom of the page and then fill out the rows starting with the top row and then going down. After each time filling out a row, select the check button to see how many you have in the right spot. All colors are used and there is no possibility of repeat colors in the code. Good luck and have fun.
